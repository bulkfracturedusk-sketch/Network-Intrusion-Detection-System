# Network Intrusion Detection System (NIDS)

## Project Overview
This project implements a Machine Learning based Network Intrusion Detection System (NIDS) using IBM watsonx AutoAI. The system analyzes network traffic data and classifies connections as either Normal or Anomalous, helping identify potential cyber attacks.

## Domain
Cybersecurity / Network Security

## Objective
To develop an intelligent Network Intrusion Detection System capable of detecting malicious network activities and distinguishing them from normal network traffic.

## Technologies Used
- IBM watsonx.ai AutoAI
- IBM Cloud
- Machine Learning
- Snap Random Forest Classifier
- Network Traffic Dataset

## Features
- Automatic intrusion detection
- Normal vs Anomaly classification
- Cloud deployment using IBM Watson Machine Learning
- Real-time prediction API
- High accuracy attack detection

## Model Performance
- Best Pipeline: Snap Random Forest Classifier
- Accuracy: 99.6%
